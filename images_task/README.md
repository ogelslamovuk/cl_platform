# Demo Posters Image Task Package

Цель пакета: получить 17 профессиональных PNG-постеров для demo-афиши `cl_platform`, но не запускать сразу весь набор вслепую.

## Режим работы

Сначала делаем только один тестовый постер:

```text
poster-mikhail-stasov-vernus-k-tebe.png
```

После визуального approve по этому тесту — запускаем генерацию остальных 16 постеров и финальную упаковку.

## Как запускать

1. Распаковать этот архив локально.
2. Открыть отдельную ChatGPT Images / image-generation сессию.
3. Загрузить все `.md` файлы из папки.
4. Первым сообщением отправить содержимое `IMAGE_INITIAL_MESSAGE.md`.
5. Получить один тестовый постер Михаила Стасова.
6. Оценить глазами: реалистичность, читаемость, не-mock, не-minimal, звездность.
7. После approve отправить содержимое `AFTER_TEST_APPROVAL_MESSAGE.md`.
8. Получить оставшиеся 16 PNG и один архив с 17 PNG.

## Ожидаемый финальный результат

Один архив:

```text
cl_platform_demo_posters_17_png.zip
```

Внутри ровно 17 PNG-файлов с именами из `DEMO_POSTER_MATRIX.md`.
