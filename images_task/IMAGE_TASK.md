# Task

## Goal

Create a final poster asset pack for the demo B2C event listing of `cl_platform`.

## Product context

`cl_platform` is a demo-first prototype of a government-oriented platform for cultural and mass events. The project is prepared for demonstrations to ministries and stakeholders. The public B2C event listing must look credible and presentation-ready.

The current demo posters look fake and repetitive. The new posters must make the event listing look like a believable cultural marketplace, not a mock interface filled with placeholder graphics.

## Approved scope

- Create poster image assets only.
- Use the exact events defined in `DEMO_POSTER_MATRIX.md`.
- Use professional poster-like styling close to real event advertising.
- Keep posters human-centered and genre-specific.
- Use a test-first workflow: one approved test poster first, then the rest.

## Out of scope

- No UI redesign.
- No code changes.
- No GitHub changes.
- No ticket mockups.
- No platform logo system.
- No new event creation outside the matrix.
- No changes to routes, cards, labels, pricing, seatmaps, admin or operator flows.

## Output

Phase 1:
- One test PNG: `poster-mikhail-stasov-vernus-k-tebe.png`.

Phase 2 after approval:
- 17 PNG files, 2:3 portrait.
- Exact filenames per matrix.
- Final result as one ZIP archive if possible.

## Success criteria

- Each poster looks like a real event poster.
- All posters are clearly different.
- All posters match event type, audience and venue status.
- No poster feels like placeholder art.
- No poster uses a repeated template.
