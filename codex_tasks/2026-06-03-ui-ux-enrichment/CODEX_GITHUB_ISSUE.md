# CODEX_GITHUB_ISSUE.md

## Title

`cl_platform`: комплексное UI/UX и demo-flow обогащение Organizer / Admin / SeatMap / B2C без поломки текущего flow

## Summary

Нужно одним автономным запуском довести текущий прототип `cl_platform` до более цельного, зрелого и демонстрационного состояния.

Это пакет правок по существующему работающему прототипу. Нельзя переписывать продукт, ломать текущую навигацию, создавать параллельные сущности или строить production backend.

Главная цель: текущая платформа должна выглядеть как осмысленная государственная операционная система рынка мероприятий, а не как набор отдельных mock-экранов.

## Business intent

Прототип готовится к демонстрации институциональным / министерским стейкхолдерам.

После выполнения задачи зритель должен понимать:

- как организатор создаёт заявку на мероприятие;
- какие документы и участники прикладываются;
- как заявка проходит проверки;
- как Центр Управления видит заявки, реестры, регионы, операторов и контроль;
- как крупные события без схемы отличаются от событий с посадочными местами;
- как билеты, операции, постеры и B2C-афиша выглядят как часть одного продукта.

## Scope

Изменения касаются только текущего frontend/demo-прототипа.

Разрешено:

- расширять существующие mock-data;
- добавлять поля в существующие demo-сущности;
- добавлять UI-состояния, фильтры, сортировки и mock-preview;
- улучшать верстку существующих экранов;
- делать click-through mock-переходы;
- добавлять mock-документы без реальных персональных данных;
- добавлять базовую региональную demo-модель;
- добавлять seatmap templates в существующий конструктор;
- менять demo-постеры и B2C-сетку.

Запрещено:

- production backend;
- реальные API МВД / миграции / платежей;
- настоящие паспортные данные;
- создание DTCM/Dubai/NEN-сущностей;
- новый независимый реестр площадок;
- второй независимый flow покупки билетов;
- broad refactor;
- слом существующего flow;
- удаление рабочих разделов;
- изменение branding без прямого требования;
- добавление новых библиотек без прямой необходимости.
