# Presentation Design Skill

Status: draft v1.

Purpose: guidance for preparing business presentations, pitch decks, commercial proposals, product vision decks, PDF decks, and executive summaries.

Core idea: a presentation should have a clear storyline, one main idea per slide, controlled text density, readable hierarchy, and business-appropriate visual style.

Main sections to expand:

- scope and use cases
- presentation registers
- storyline patterns
- slide density rules
- title rules
- visual hierarchy
- typography
